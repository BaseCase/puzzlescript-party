Inspiration

I showed you how to make a basic Sokoban clone, but PuzzleScript can do a lot more than that! Take a look at three of my favorite PuzzleScript games if you want to be inspired by the creativity and ingenuity of some great game designers.

Heroes of Sokoban

by Jonah Ostroff

Heroes of Sokoban feels like the game that PuzzleScript was made to build. It has some clever twists on the original Sokoban formula, and combines simple rules in surprising and complex ways.

Skipping Stones to Lonely Homes

by Alan Hazelden

Skipping Stones to Lonely Homes shows how far you can push PuzzleScript with a little creativity. When I first played this, I didn’t believe it was possible in the engine. Studying the code taught me a ton about more advanced PuzzleScript development.

Aunt Flora’s Mansion

by anna anthropy

Aunt Flora’s Mansion is an awesome example of a PuzzleScript game with one interconnected world. If you’d like to make a game that’s a single large map rather than several small levels, this is the game to study.
